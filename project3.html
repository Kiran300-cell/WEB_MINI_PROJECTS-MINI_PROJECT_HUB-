Weather Monitoring Dashboard

A responsive environmental dashboard interface showing live weather data.

This project presents a clean, minimalistic UI for displaying data such as temperature, humidity, pressure, rainfall, and soil moisture.

Designed using only HTML and CSS, the layout uses liquid grids, flex-based components, and subtle micro animations. The dashboard is ideal for IoT systems, school mini-projects, and sensor-based prototypes.
